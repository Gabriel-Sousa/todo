## 🚀 Tecnologias

Tecnologias utilizadas no projeto:

- ReactJS
- Vite
- Typescript
- Styled Components

<br>

## 💻 Projeto

O ToDo é uma aplicação para criar uma lista de tarefa, atualizar status para concluído e remover alguma tarefa.

<br>

## ⚒️ Features

- [x] Cadastro de tarefa
- [x] Checar a tarefa
- [x] Remover Tarefa
- [x] Contator de Tarefa

<br>
